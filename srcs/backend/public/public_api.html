Public API Tester

This page sends requests directly to the backend and includes the x-api-key header automatically.

There is a rate limit applied to the public API. If you test several endpoints quickly, expect a 429 Rate limit exceeded response.

Connection

GET

List Quizzes

Returns quiz IDs and titles. Good first endpoint to confirm the API key works.

GET

Show Quiz

Returns one quiz and its questions, but leaves out the correct answers.

GET

Show User

Returns a public user profile with id, userName, and avatarUrl.

POST

Create Quiz Session

Creates a quiz session. Required body fields are quizId and hostId.

PUT

Update User

Changes a username. The name must be unique or the API returns a 400.

DELETE

Delete Quiz Session

Deletes an existing quiz session by ID.
