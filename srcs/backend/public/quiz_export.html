Quiz Export And Import Tester

This page targets the quiz admin and export endpoints under /api/admin/quiz/:id.

Use it to inspect full quiz data, export structured JSON or CSV, and import quizzes from CSV or JSON files.

Connection

EXPORT

Export JSON

Returns a JSON export with quiz metadata, all questions, correct answers, and aggregated answer statistics.

EXPORT

Export CSV

Returns CSV with one row per question, including total answers, correct answers, incorrect answers, and correct rate.

IMPORT

Import CSV

Uploads a CSV file to /api/quizzes/import-csv. The multipart field name is csv.

IMPORT

Import JSON

Uploads a JSON file to /api/quizzes/import-json. It accepts both import payloads and exported quiz JSON.

Route Summary

GET /api/admin/quiz/:id/export?format=json

Structured export payload, suitable for integrations or debugging.

GET /api/admin/quiz/:id/export?format=csv

Spreadsheet-friendly export with per-question stats.

POST /api/quizzes/import-csv

Imports a quiz from a CSV file uploaded as multipart form-data with the field name csv.

POST /api/quizzes/import-json

Imports a quiz from a JSON file uploaded as multipart form-data with the field name json.
