WS Chat Tester

Dedicated test page for the token-auth websocket chat flow. It connects to /ws?token=..., lets you join or leave a conversation, send chat events, inspect raw socket frames, and hit the related REST endpoints with the same bearer token.

Connection

Disconnected

The websocket is opened with ?token=YOUR_TOKEN. REST calls use Authorization: Bearer YOUR_TOKEN.

Conversation

If the socket is not in the room you should now only receive Not joined to conversation, not the extra unknown-event error.

REST Assist

Use this block to create a conversation or inspect existing ones without leaving the page.
